Program Name

Anti-Theft System SMS Service

Program Description

The Anti-Theft System SMS Service is an SMS notification and command-reply service for a personal vehicle anti-theft device operated by a single owner. The system sends automated text messages related to device status, security events, and command confirmations.

Message Types

You may receive the following types of messages:

  • Intrusion alerts — vibration/door-open events with GPS coordinates and photo link
  • Status replies — system armed/disarmed, GPS location, battery level
  • Command confirmations — ARM, DISARM, STATUS, PHOTO, GPS, IMMOBILIZE, RESTORE

Message Frequency

0–2 messages per week during normal operation, plus event-driven alerts when device sensors detect intrusion attempts.
